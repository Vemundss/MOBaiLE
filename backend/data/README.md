# Runtime Data

This directory is for local runtime state generated by the backend.

It is distinct from repo-owned runtime defaults under `.mobaile/runtime/`.

Examples:
- `runs.db`
- `profiles/<profile_id>/AGENTS.md`
- `profiles/<profile_id>/MEMORY.md`
- `sessions/<session_id>/AGENTS.md`
- `sessions/<session_id>/MEMORY.md`

These files are intentionally ignored by git and should not be committed to the public repository.
